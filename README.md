## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or pnpm

### Installation

1. Clone the repository

```sh
git clone https://github.com/1amJoseph/Portfolio.git
cd Portfolio
```

2. Install dependencies

```sh
npm install
```

3. Start the development server

```sh
npm run dev
```

4. Open your browser at `http://localhost:4321`
